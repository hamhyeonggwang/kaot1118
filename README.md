# 한국작업치료사협회 (KAOT) 키오스크 랜딩 페이지

작업치료를 소개하는 고급스러운 웹페이지입니다.

## 주요 기능

- 🎨 **모던한 UI/UX**: 글래스모피즘 효과와 부드러운 애니메이션
- ✨ **인터랙티브 요소**: 호버 효과, 파티클 배경, 그라데이션 효과
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- 🎭 **애니메이션**: 로고 전환, 버튼 호버, 모달 애니메이션
- 🌈 **시각적 효과**: 그라데이션, 그림자, 블러 효과

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React (아이콘)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 주요 개선 사항

1. **고급 애니메이션**
   - 로고 페이드 인/아웃 효과
   - 버튼 스케일 및 회전 효과
   - 모달 부드러운 전환

2. **시각적 효과**
   - 파티클 배경 애니메이션
   - 글래스모피즘 디자인
   - 그라데이션 및 그림자 효과
   - 호버 시 빛나는 효과

3. **사용자 경험**
   - 버튼 호버 시 설명 표시
   - 모달 상세 정보 제공
   - 반응형 레이아웃

4. **디자인 개선**
   - 더 풍부한 색상 팔레트
   - 개선된 타이포그래피
   - 일관된 디자인 시스템
